# HUL_DCF
A Discounted Cash Flow (DCF) valuation model of Hindustan Unilever Ltd. (HUL) built in Microsoft Excel. This project includes financial forecasting, FCFF calculation, WACC, terminal value, and company valuation using real financial data.
