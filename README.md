# HUL DCF Valuation Model

## Project Overview
This project is a Discounted Cash Flow (DCF) valuation model built for Hindustan Unilever Ltd. (HUL).  
The model forecasts the company’s financial performance till 2031 using historical financial statements and valuation assumptions.

---

## Key Features
- Revenue Forecasting
- EBIT Margin Analysis
- FCFF (Free Cash Flow to Firm) Modeling
- Capex & Depreciation Forecasting
- Working Capital Analysis
- Tax Rate Normalization
- Financial Statement Linking
